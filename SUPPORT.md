# Support

Support for this project is provided on a best-effort basis and is not guaranteed.

## How to get help

1. **Open an Issue (preferred)**  
   Please submit questions, bug reports, and feature requests by opening a GitHub Issue in this repository. When reporting a bug, include steps to reproduce, expected vs. actual behavior, and relevant logs or configuration details (redacted as needed).

2. **Email**  
   If you cannot share details publicly (e.g., sensitive information), contact the maintainers by email:
   - katya@irradiant.tech  
   - lisa.gavronskiy@irradiant.tech

## Response expectations

We review incoming requests as time permits. Response times may vary, and we may not be able to respond to every request.

## Security issues

If you believe you have found a security vulnerability, please **DO NOT** open a public Issue. Instead, report it by email to the addresses above.
